# android-ndk-harfbuzz
Usage:
```
1. Install android ndk

2. Download tarball:
	cairo-1.17.2.tar.xz
	freetype-2.10.2.tar.xz
	gettext-0.21.tar.gz
	glib-2.56.4.tar.xz
	harfbuzz-2.7.2.tar.xz
	icu4c-67_1-src.tgz
	libffi-3.3.tar.gz
	libiconv-1.16.tar.gz
	libpng-1.6.37.tar.xz
	pcre-8.44.tar.bz2
	pixman-0.40.0.tar.gz
	zlib-1.2.7.tar.gz

3. Extract tarballs

4. ./build_all.sh
```
